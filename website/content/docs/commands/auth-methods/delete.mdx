# auth-methods delete

Command: `boundary auth-methods delete`

The `auth-methods delete` command deletes an auth method given its ID.

## Example

The following example deletes an auth method with the ID, `moidc_oHt4HQFCrN`.

```shell-session
$ boundary auth-methods delete -id amoidc_oHt4HQFCrN
```

## Usage

<CodeBlockConfig hideClipboard>

```shell-session
$ boundary auth-methods delete [options] [args]
```

</CodeBlockConfig>

### Command options

- `-id` `(string: "")` - The ID of the auth method to delete.
