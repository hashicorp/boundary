Configure a basic target for testing

- Configure target with public address or deploy a worker
- Connect to target using CLI/Desktop app

Connect to a target using my method of choice

- Connect using the CLI
- Customize my CLI connection workflow
- Search available hosts and connect to a target

Seamlessly connect to targets
- Set up transparent sessions using the Desktop app
- Set up transparent sessions using the CLI
- Mange the cache daemon
